O této anketě

Školní projekt zaměřený na tvorbu jednoduché hlasovací aplikace s trvalým ukládáním dat a sdílenými výsledky mezi všemi uživateli.

Co aplikace umí

Hlasování s uložením do databáze
Sdílené výsledky v reálném čase
Reset chráněný tokenem
Data přežijí reload i restart serveru

Technologie

Backend: Python 3 + Flask
Databáze: SQLite
Hosting: PythonAnywhere
Frontend: HTML + CSS, žádný JavaScript
